How to use?                                                                                    
                                                                                               
Options :                                                                                      
    -c (--command=) :  Runs the commands written below                                         
        help :  Get to help : python3 kripter.py -c help                                       
        hashes :  Show supported encryption/decryption algorithms                              
        info :  Get to software information : python3 kripter.py -c info                       
        framework:  Get to kripteks-framework : python3 kripter.py -c framework                
        settings :  Get to settings : python3 kripter.py -c settings                           
                                                                                               
                                                                                               
    Usage :                                                                                    
    python3 kripteks.py -c help                                                                
    or  python3 kripteks.py --command=help                                                     
                                                                                               
Encryption :                                                                                   
    -e (--encrypt=) :  write the hash algoritm. (md5,sha1,sha224 and ... [-c hashes])          
    -s (--string=) :  Add the word you want to encrypt here.                                   
    -f (--file=) :  Add the text file you want to encrypt here. (This is supported only .txt)  
                                                                                               
    Usage :                                                                                    
    python3 kripteks.py -e [algorithm] -s [string]                                             
    python3 kripteks.py -e [algorithm] -f [file.txt]                                           
                                                                                               
Decryption :                                                                                   
    -d (--decrypt=) :  write the hash algoritm. (md5,sha1,sha224 and ...[-c hashes])           
    -h (--hash=) :  Add encrypted text you want decrypt here.                                  
    -f (--file=) :  Add your wordlist (This is supported only .txt)                            
                                                                                               
    Usage :                                                                                    
    python3 kripteks.py -d [algorithm] -h [hashed_string] -f [wordlist.txt]                    
    or python3 kripteks.py --decrypt=[algorithm] --hash=[hashed_string] --file=[wordlist.txt]
